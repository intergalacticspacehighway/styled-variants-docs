# createTheme

- `createTheme` function is a helper function to create theme. It returns with utilities that can be used within the application.

#### Import

```jsx
import { createTheme } from "react-native-styled-variants";
```

#### Usage

```jsx
import { createTheme } from "react-native-styled-variants";
const theme = {
  space: {
    1: 4,
    1.5: 6,
    2: 8,
    2.5: 10,
    3: 12,
    3.5: 14,
    4: 16,
  },
  fontSizes: {
    sm: 14,
    md: 16,
    lg: 18,
    xl: 24,
    "2xl": 28,
    "3xl": 32,
    "4xl": 48,
  },
  colors: {
    primary: "#000",
  },
};

const breakpoints = {
  base: 0,
  sm: 480,
  md: 768,
  lg: 992,
  xl: 1280,
};

export const { createVariant, ThemeProvider, useTheme, useCurrentBreakpoint } =
  createTheme({
    theme,
    breakpoints,
  });
```

## ThemeProvider

- `createTheme` util returns a `ThemeProvider` that should be wrapped over the entire application.

```jsx
import { ThemeProvider } from "./theme";

export default const App = () => {
  return <ThemeProvider>{/* Your app goes here */}</ThemeProvider>;
};
```

## useTheme

- `createTheme` util returns a `useTheme` hook that includes a theme setter and theme object.
- This hook won't be needed for most of the cases, but it's there if you need to access theme tokens in a component.

```jsx
import { useTheme } from "./theme";

export default const App = () => {
  const { theme, setTheme } = useTheme();
};
```

## useCurrentBreakpoint

- `createTheme` util returns a `useCurrentBreakpoint` hook that includes current breakpoint from the breakpoints passed in `createTheme` function.
- This hook won't be needed for most of the cases, but it's there if you need to access current breakpoint or a helper function `resolveResponsiveValue`.

```jsx
import { useCurrentBreakpoint } from "./theme";

export default const App = () => {
  const {resolveResponsiveValue, currentBreakpoint} = useCurrentBreakpoint();
};
```
