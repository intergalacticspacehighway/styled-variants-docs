# SX prop

- `sx` prop can be used to write inline styles with responsive and token values. It is memoized into StyleSheet.create using `useMemo` if it doesn't contain a variable. `sx` prop can be used in any component that supports `style` prop.

## Usage

```jsx
import { View } from "react-native";

const Component = () => {
  <View
    sx={{
      padding: "$space.10",
      borderRadius: 10,
      backgroundColor: "$colors.primary",
    }}
  />;
};
```

## Responsive tokens

Responsive breakpoints passed in `createTheme` can be used as below. Note the `@` prefix is required.

```jsx
import { View } from "react-native";

const Component = () => {
  <View
    sx={{
      padding: { "@base": "$space.8", "@sm": "$space.10", "@md": "$space.16" },
      borderRadius: 10,
      backgroundColor: "$colors.primary",
    }}
  />;
};
```

<br />

**Note:** Alongwith `sx`, `contentContainerSX` is supported that is equivalent of `contentContainerStyle`.

<br />

## Typescript support

- Create a file named `extensions.d.ts` in your typescript project and copy paste the below content. `ITheme` and `IBreakpoints` are type of your theme and breakpoints passed in [createTheme](/create-theme).

```jsx
import "react-native";
import { RNStyles } from "react-native-styled-variants";
import type { ITheme, IBreakpoints } from "./theme";

declare module "react-native" {
  interface ViewProps {
    sx?: RNStyles<ITheme, IBreakpoints>;
  }
  interface ScrollViewProps {
    contentContainerSX?: RNStyles<ITheme, IBreakpoints>;
  }
  interface ImageProps {
    sx?: RNStyles<ITheme, IBreakpoints>;
  }
  interface TextProps {
    sx?: RNStyles<ITheme, IBreakpoints>;
  }
}

```
