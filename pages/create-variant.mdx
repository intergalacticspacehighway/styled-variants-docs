# createVariant

Create variant provides an interface to define base and variant styles for a component.

<br />

## Base style

`createVariant` accepts all the React Native styles and more as the second argument. Theme tokens will be suggested if you're using TypeScript.

```js
const Button = createVariant(Pressable, {
  padding: "$space.10",
});
```

## Variants

- Variants can be specified in the variants object. Here, we're defining a variant named primary and it's value as true.

```js
const Button = createVariant(Pressable, {
  padding: "$space.10",
  borderRadius: 10,
  variants: {
    primary: {
      true: {
        backgroundColor: "$colors.primary",
      },
    },
    size: {
      lg: {
        padding: "$space.12",
      },
    },
  },
});

// Usage
<Button primary size="lg" />;
```

## Interaction styles

- `_hovered`, `_pressed`, `_focused` styles can be specified in `createVariant` API

```js
const Button = createVariant(Pressable, {
  padding: "$space.10",
  variants: {
    primary: {
      true: {
        backgroundColor: "$colors.primary",
      },
    },
  },
  _hovered: {
    backgroundColor: "$colors.hover",
  },
});
```

## Responsive styles

- We can define responsive values using `@` tokens.

```js
const Button = createVariant(Pressable, {
  padding: { "@base": "$space.10", "@sm": "$space.8" }
  variants: {
    primary: {
      true: {
        backgroundColor: "$colors.primary",
      },
    },
  },
  _hovered: {
    backgroundColor: "$colors.hover",
  },
});
```
